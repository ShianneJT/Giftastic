# GifTastic

A web based application that uses the GIPHY API to display the gifs based on the topic of the user's choice.

- [https://shiannejt.github.io/Giftastic/]("https://shiannejt.github.io/Giftastic/")

## Technologies Used

- AJAX
- GIPHY API
- jQuery
- JavaScript
- Bootstrap
- CSS
- HTML
